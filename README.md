# CRUD API Assignment

This project is a simple REST API built using Node.js, Express, and SQLite.
It demonstrates all CRUD operations on a database.

## Routes

GET /items  
GET /items/:id  
POST /items  
PUT /items/:id  
DELETE /items/:id  

## Example JSON Body

{
  "name": "Notebook",
  "quantity": 5
}

## How to Run

1. Install dependencies
   npm install

2. Start the server
   npm start

3. Test using Postman at:
   http://localhost:3000
